# Minecraft Bot с использованием Mineflayer

Бот для Minecraft, который автоматически подключается к серверу и выполняет действия:
- Прыжки
- Вращение вокруг себя
- Атака рукой

## Требования

- Node.js 16.0.0 или выше
- Сервер Minecraft (локальный или удаленный)

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Настройте конфигурацию в файле `config.js`:
   - Укажите адрес сервера (`host`)
   - Укажите порт сервера (`port`)
   - Укажите версию Minecraft (`version`)
   - Настройте имя бота и параметры действий

## Запуск

### Универсальный скрипт (автоопределение ОС):
```bash
chmod +x start.sh
./start.sh
```

### Для Android Termux:
```bash
chmod +x start-termux.sh
./start-termux.sh
```
[Подробная инструкция для Termux](INSTALL-TERMUX.md)

### Для Linux/macOS:
```bash
chmod +x start-linux.sh
./start-linux.sh
```
[Подробная инструкция для Linux](INSTALL-LINUX.md)

### Для Windows:
```bash
start.bat
```

### Основной бот:
```bash
npm start
```
Или:
```bash
node bot.js
```

### Локальное тестирование:
```bash
node bot-local.js
```

### Проверка сервера:
```bash
node check-server.js
```

## Конфигурация

### Основная конфигурация (`config.js`):
```javascript
server: {
  host: 'zuboklyak.aternos.me',    // Адрес сервера Aternos
  port: 58899,                     // Порт сервера (уточните в панели Aternos)
  version: '1.21.11'               // Версия Minecraft
},
bot: {
  username: 'AutoBot',  // Имя бота
  actions: {
    jumping: { enabled: true, interval: 1000 },
    spinning: { enabled: true, interval: 500, speed: 45 },
    attacking: { enabled: true, interval: 1500, range: 4 }
  }
}
```

### Локальная конфигурация (`config-local.js`):
Для тестирования на локальном сервере.

## Действия бота

1. **Прыжки**: Бот прыгает с заданным интервалом
2. **Вращение**: Бот вращается вокруг своей оси
3. **Атака**: Бот атакует ближайших мобов или бьет воздух

## Управление

### Команды в консоли:
- `stop` или `exit` - остановить бота
- `reconnect` - принудительное переподключение
- `status` - показать статус бота
- `say <текст>` - отправить сообщение в чат Minecraft
- `help` - показать справку по командам

### Автоматические функции:
- **Автопереподключение** - при потере соединения бот пытается переподключиться каждые 10 секунд
- **Автоответ в чат** - бот автоматически отвечает на приветствия
- **Минимальный вывод** - по умолчанию вывод действий отключен (можно включить в коде)

## Настройка отладки

В коде можно включить/выключить вывод действий бота:
```javascript
const DEBUG_ACTIONS = false; // false - убрать вывод, true - показывать
```

## Примечания

- Для работы с оффлайн-серверами используется аутентификация `offline`
- Версия 26.1.2, указанная в задании, вероятно, относится к версии протокола
- Для Minecraft 1.20.4 используется протокол ~765
